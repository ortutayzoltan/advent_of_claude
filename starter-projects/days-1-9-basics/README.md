# Advent Project

A learning project for mastering Claude Code - an AI-powered development assistant.

## Project Overview

This project serves as a hands-on playground for learning Claude Code features through the "Advent of Claude Code" 25-day curriculum.

## Features

- Simple greeting functionality
- Command-line interface support
- Modular code structure

## Getting Started

```bash
# Create and activate virtual environment (optional but recommended)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Run the main script
python hello.py

# Run with custom name
python hello.py --name "Your Name"
```

## Project Structure

```
days-1-9-basics/
├── README.md          # This file
├── hello.py           # Main entry point
├── utils.py           # Utility functions
├── CLAUDE.md          # Claude Code project configuration
└── requirements.txt   # Python dependencies
```

## Learning Path

This starter is designed for Days 1-9 of the Advent of Claude Code:

- **Day 1**: Start a session, create README
- **Day 2**: Use `@` file references
- **Day 3**: Run commands with `!`
- **Day 4**: Explore `/help`
- **Day 5**: Initialize with `/init`
- **Day 6**: Git commits
- **Day 7**: Codebase exploration
- **Day 8**: Multi-file edits
- **Day 9**: Context management with `/clear`
